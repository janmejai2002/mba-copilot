## üî¨ Understanding the Algorithm

### How Connections Are Made:
1. **Word Similarity**: Concepts sharing keywords get linked
2. **Chronological Flow**: Sequential concepts from lecture are connected
3. **Strength Calculation**: More shared words = stronger link = thicker line

### Node Positioning:
- **Force Simulation**: Nodes push/pull based on connections
- **Natural Clustering**: Related concepts group together
- **Physics-Based**: Graph self-organizes for clarity

### Learning Path Logic:
1. Count connections for each concept
2. Sort by importance (connection count)
3. Respect chronological order for equal importance
4. Result: Foundational concepts first, advanced topics later

## üêõ Troubleshooting

**Graph looks cluttered:**
- Use search/filter to focus on specific topics
- Zoom in to see details
- Drag nodes apart for better spacing

**Can't find a concept:**
- Check if search is active (clear search box)
- Verify category filter is set to "All"
- Concept might not have been extracted yet (wait for AI processing)

**Export not working:**
- Check browser download permissions
- Try different browser if issue persists
- File saves as `study-guide.md` in Downloads folder
